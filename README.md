# Требования к системе
Для запуска проекта требуется:
* PHP ≥ 8.0
* PostgreSQL ≥ 11
* Веб-сервер (Apache / Nginx / OpenServer / OSPanel)
* Включённое расширение PHP:
    * pdo
    * pdo_pgsql
* Поддержка JavaScript в браузере

# Инструкция по установке
1. Скопировать проект в папку веб-сервера
2. Открыть файл `setup.php` для создания базы данных
3. Открыть главную страницу проекта `index.php`
4. Данные для входа администратора `Логин: admin Пароль: 123`
